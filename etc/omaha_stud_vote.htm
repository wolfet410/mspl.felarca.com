Omaha/7-Card Stud Vote

After the Omaha tournament on February 11th and the 7-Card Stud tournament on March 11th, there will be a vote to determine if the league will continue to run each tournament type and, if so, how many. The questions that will be posed are below. Only players that participate in the tournament can vote on the tournament's future within the league. If a tournament is voted on to be removed or reduced, removed occurances of that tournament will be replaced with Texas Hold'em. Currently there are 4 occurances of Omaha and 3 occurances of 7-Card Stud on the schedule.

Note: The following questions will be worded the same for 7-Card Stud except for the game type named and the number of tournaments.

1. Should the league continue to Omaha tournaments?

  • Yes
  • No

2. If so, should the number of Omaha tournaments be reduced?

  • Yes
  • No

3. If so, how many Omaha tournaments should there be?

  • 3
  • 2
  • 1
